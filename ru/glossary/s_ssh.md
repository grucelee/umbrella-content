# SSH

SSH (или Secure SHell) - это метод, позволяющий вам безопасно управлять удаленным компьютером через интерфейс командной строки. Одной из особенностей протокола SSH является то, что он хорош для отправки команд; вы также можете использовать его для надежной ретрансляции интернет-трафика между двумя компьютерами. Чтобы настроить ссылку ssh, удаленная система должна работать как ssh-сервер, а вашей локальной машине нужна клиентская программа ssh.
