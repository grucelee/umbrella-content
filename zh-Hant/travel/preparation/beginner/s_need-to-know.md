# 誰*真的*需要知道？

告訴更少的人您的旅行有助於保持更安全。但旅行準備通常涉及共享信息。

在需要了解政策的同時平衡安全性和實用性。

（在[管理信息](umbrella://information/managing-information)課程中了解更多相關信息。）

您為旅行準備的信息可能包括：

*   戰略，境況和風險評估;
*   關鍵文件和醫療信息的副本;
*   通信方式和聯繫方式;
*   行程和其他標準操作程序;
*   安全和緊急協議。

對於每個項目，請詢問：

*   誰需要了解這些信息？
*   他們將如何保護它？
